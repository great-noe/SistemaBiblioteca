## Descripción
Sistema de gestión bibliotecaria diseñado para administrar el flujo de préstamos, devoluciones y reservas, con control de inventario de libros, gestión de usuarios y registro de multas.

## Tecnologías
- .NET
- Windows Forms
- MySQL
- Entity Framework

## Módulos Principales
1. Gestión de Autores y Libros
2. Gestión de Usuarios
3. Gestión de Préstamos
4. Gestión de Reservas
5. Gestión de Multas

## Estado del Proyecto
Día 1 - Estructura base creada. Arquitectura de 4 capas (Entidades, Datos, Negocio, Presentación) implementada y lista para el desarrollo de la lógica interna.
